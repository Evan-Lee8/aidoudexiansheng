# 视觉闹钟应用与俄罗斯方块游戏

## 1. 视觉闹钟应用 (Visual Alarm Clock)

### 功能特点
- 直观的视觉时钟显示
- 可设置多个闹钟
- 闹钟响铃时显示醒目的视觉效果
- 支持自定义闹钟铃声
- 简单易用的用户界面

### 使用方法
1. 下载 `dist/visual_alarm_clock.exe` 文件
2. 双击运行应用程序
3. 在界面上设置闹钟时间
4. 点击"添加闹钟"按钮保存
5. 闹钟响铃时，会显示视觉提醒并播放铃声

### 系统要求
- Windows 7/8/10/11
- .NET Framework 4.5 或更高版本

## 2. 俄罗斯方块游戏 (Tetris Game)

### 功能特点
- 经典的俄罗斯方块游戏玩法
- 实时分数、行数和等级显示
- 下一个方块预览功能
- 支持键盘和鼠标操作
- 响应式设计，适配不同屏幕尺寸

### 操作说明
- **键盘控制**：
  - 方向键 ← →：左右移动方块
  - 方向键 ↑：旋转方块
  - 方向键 ↓：加速下落
  - 空格键：立即下落
- **鼠标控制**：
  - 使用界面上的按钮控制游戏

### 运行方法
1. 打开 `tetris.html` 文件
2. 游戏将在浏览器中自动开始
3. 点击"开始游戏"按钮开始游戏
4. 点击"暂停"按钮暂停游戏
5. 点击"退出"按钮退出游戏
